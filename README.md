# Mon Premier Projet : *Site Vitrine La Gastro*
Ceci est le projet frontend de mon site vitrine nommée **La Gastro**.

## Démo en ligne 
Pour voir le site fonctionner en direct sans télécharger le code, cliquez sur le lien ci-dessous :
** [Voir la Démo Live](https://christ-moz.github.io/Sitevitrine/) **

## Technologies utilisés
- HTML5
- CSS3 (Mise en page, et responsive design)
- JavaScript (Interactivité)

## Structure du Projet
- `index.html` : Page principale du site
- `style.css` : Styles et design
- `script.js` : Logique et interactions
- `/images` : Dossier contenant les visuels du site
